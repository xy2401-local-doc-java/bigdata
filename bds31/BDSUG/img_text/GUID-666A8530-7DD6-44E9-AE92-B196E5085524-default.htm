The figure has a dimension table and a fact table. The dimension table has a product column with fan, hard drive, LED display, and power cord, and quantity column, labeled by M, with the quantities for each product. The M column has the quantities 1, 3, 5, and 8. The fact table has columns labeled A, M, C, and D. The M column has the numbers 1, 3, 5, 5, 8, and 5. The first region index is queried using the bloom filter. The second region index is not queried because the value for the WHERE statement is not met.
